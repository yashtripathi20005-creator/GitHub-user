# GitHub Profile Search Tool

A simple web application to search and display GitHub user profiles with their repositories.

## Features
- Search any GitHub user by username
- Display profile info (bio, location, company, stats)
- Show latest repositories with stars, forks, and language
- Responsive design for all devices

## Quick Start

```bash
# Clone and install
git clone <your-repo>
cd github-profile-search
pip install -r requirements.txt

# Run
python app.py
